Разбери поисковый запрос исследователя по горно-металлургии.
Верни только JSON без пояснений.

Запрос:
{query}

Формат JSON:
{{
  "intent": "literature_search|parameter_search|comparison|overview",
  "entities": [{{"type": "Material|Process|Equipment", "name": "..."}}],
  "filters": {{
    "geo": ["RU", "foreign"],
    "year_from": null,
    "year_to": null,
    "numeric": [{{"parameter": "сульфаты", "operator": "lte", "value": 300, "unit": "мг/л"}}]
  }},
  "search_text": "краткая строка для семантического поиска"
}}

Правила:
- geo: RU для отечественной практики, foreign для зарубежной, world для мировой
- operator только lte, lt, gte, gt, eq
- search_text на русском, без лишних слов
- Если запрос структурирован (строки «Материал:», «Процесс:», «Условия:», «География:», «Период:»), извлеки сущности и фильтры из этих полей
- Условия с числами (≤, ≥, мг/л, °C и т.п.) занеси в filters.numeric
- Последняя строка-вопрос используй как intent; search_text собери из материала, процесса и ключевых условий
